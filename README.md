# Portfolio

A modern and minimal developer portfolio built using React shadcn/ui, and Tailwind CSS.

## 🚀 Tech Stack

* ⚛️ React + TypeScript
* 🎨 Tailwind CSS
* 🧩 shadcn/ui
* ⚡ Vite for fast development
* 🌙 Dark mode support
* 💡 Responsive & accessible design

## 📦 Getting Started

### Requirements

* Node.js (v18+)
* npm or yarn

### Setup

```bash
git clone https://github.com/your-username/portfolio.git
cd portfolio
yarn install
yarn dev
```
